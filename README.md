## what

a bulk qr code generator. put in text you want qr of (one per line) and it'll throw a zip with all the qrs at you.

## why

i couldn't find a simple to use bulk qr code generation software that didn't make the non-technical guys at my workplace pull their teeth out, so i made this to link to them.

## how

you have two options:

- go to https://theabhishek2511.github.io/qr
- download [`index.html`](https://raw.github.com/theabhishek2511/qr/main/index.html) (right click link->save link as) and open it in a browser - it can run completely offline.
